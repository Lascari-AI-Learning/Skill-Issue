SPEAKER NOTES:

So what IS a skill? It's opinionated markdown files - structured documentation designed for agents to understand, but still human-readable too.

The magic is self-discovery. Agents can find and acquire processes on-demand without you manually loading context. They pull in what they need, when they need it.

And this is becoming the standard. Anthropic released this structure with Claude Code, and now many AI coding providers have adopted it as the way to structure agentic documentation.
